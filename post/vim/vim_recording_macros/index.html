使用 vim Recording 功能录制宏

录制一个宏保存到寄存器中

vim Recording 是一个很强大的功能，可以录制一些宏保存到 a-z 26个寄存器中

在 normal mode 中按 q 开始录制，然后按下 a-z 中的任意一个案件保存到该命名寄存器中

然后开始录制你的操作，最后按 q 结束

调用该宏

1
2
3

<number>@<register> "number代表要执行的次数 默认一次，register 代表一个指定的寄存器
@a "执行寄存器 a 中的宏
@@ "再次执行

删除宏

qaq ：删除 a 寄存器中的宏 or

:call setreg('a', '')

:let @a = ''

修改宏

如修改寄存器 a 中的宏

可以在任何地方，用任何方式修改

如在 cmd line mode 中修改：

ctrl+r a 将 a 寄存器中的内容复制出来，在内容前后加上'，然后修改完毕后，在前面加上 let @a= 回车即可

附加内容直接使用大写字母即可，如 ：qA 然后添加操作后按 q 结束即可

查看宏

:reg a
